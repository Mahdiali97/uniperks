## Interaction Patterns

### Button States
- **Default**: Orange bg, white text, 0 elevation
- **Hovered**: Slightly darker orange
- **Pressed**: Visual feedback, ripple effect
- **Disabled**: Grey bg, lighter text

### Input States
- **Enabled**: Grey border (grey[300])
- **Focused**: Orange border (2px), orange icon
- **Error**: Red border, error message
- **Filled**: White bg, border color changes

### Card Interactions
- **Normal**: 1px grey[200] border
- **Hovered**: Slight scale/lift
- **Selected**: Orange border or highlight
- **Disabled**: Greyed out
